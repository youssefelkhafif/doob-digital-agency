# Doob Digital Agency Landing Page 🌐

A responsive digital agency landing page built using **HTML** and **CSS**.

---

## 📌 About The Project

**Doob Digital Agency** is a front-end landing page project created to practice building a modern agency website interface.

The project includes a responsive navigation bar, hero section, service cards, contact form, and footer.  
It focuses on clean HTML structure, custom CSS styling, responsive layout, and simple user interface design.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- Font Awesome
- Responsive Design

---

## ✨ Features

- Responsive navigation bar
- Mobile menu using HTML checkbox technique
- Hero section with image illustration
- Agency-style landing page layout
- Services section
- Contact form section
- Footer with social media icons
- Organized assets folder
- Custom CSS design
